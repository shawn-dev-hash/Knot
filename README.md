#Knot

Knot is a modern desktop link manager built with Python.

It allows categorized link management with a dark UI and a QR-based phone companion mode.

# Features
- Category filtering
- Favicon caching
- Local Flask server for mobile input
- QR code access from phone
- Packaged as standalone Windows EXE

# Built With
- Python
- CustomTkinter
- Flask
- PyInstaller

# How It Works
The desktop app runs a local Flask server.
A QR code allows mobile devices on the same network to add links remotely.
Links are stored in a local JSON file.
